# coyote-snake
An example game built with [coyote-ecs](https://github.com/linuxy/coyote-ecs), a fast and simple zig native ECS.

```git clone --recursive https://github.com/linuxy/coyote-snake.git```

To build:
* zig build mimalloc
* zig build
